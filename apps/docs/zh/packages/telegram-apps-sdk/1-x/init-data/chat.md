# `Chat`

### `id`

Type: `number`

该聊天的唯一标识符。

### `photoUrl`

Type: `string`, _optional_

聊天照片的 URL。  照片可以是 .jpeg 或 .svg 格式。
仅返回从附件菜单启动的小程序。

### `type`

Type: `'group' | 'supergroup' | 'channel' | string`

聊天类型。

### `title`

Type: `string`

聊天标题

### `username`

Type: `string`, _optional_

聊天用户名。
