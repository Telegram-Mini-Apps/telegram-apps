# Поведение закрытия

На данный момент ситуация, при которой пользователь случайно закрывает
приложение при его скроллинге вверх, периодически возникает. Это
обусловлено тем, что в какой-то момент пользователь может достичь
начала страницы, после чего неосторожным движением сдвинет шторку
с приложением вниз.

Telegram Mini Apps предоставляет разработчику возможность
запросить подтверждение закрытия приложения у пользователя и убедиться,
что он действительно хочет это сделать.

:::tip

Этот функционал также можно использовать
в случае, когда в приложении происходит заполнение каких-либо форм, данные
в которых не хотелось бы случайно потерять.

:::
